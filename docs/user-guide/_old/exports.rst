Export tables
=============

MCNV2 outputs: - annotated CNV table - inheritance table - MP summary tables - Mendelian errors (candidate de novo CNVs)

--------------

Candidate de novo CNVs
----------------------

Remaining Mendelian errors after optimized filtering contain: - true de novo CNVs (biologically valid) - residual technical false positives

These are exported for downstream validation (e.g., IGV inspection, orthogonal assays).
